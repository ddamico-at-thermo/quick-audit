**Quick Audit** is a tool to speed the workflow of checking a list of URLs for target elements, with a split-screen view of the browser window. 


This is an accessible version with browser **voice control** toggle option.

Simply toggle on the Voice mode on the left sidebar and use the preset commands to navigate the workflow in Yes/No mode: 
**Yes** to mark the entry with a 'yes' in the action column and auto-advance to next entry. 
**No** to mark the entry with a 'no' in the action column and auto-advance to next entry.
**Scroll** to begin a brisk auto-scroll of the page frame. 
**Pause** to stop the auto-scroll command. 
**Next** to advance to next entry. 

When using the custom notes workflow: 
**Scroll** to begin a brisk auto-scroll of the page frame. 
**Pause** to stop the auto-scroll command. 
**Next** to advance to next entry. 
To add a **voice note**, simply start saying the note aloud and the tool will record it at the top right of the window. No command is necessary to trigger this feature, aside from toggling to "Notes" workflow mode.
