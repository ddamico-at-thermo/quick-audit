**Quick Audit** is a tool to speed the workflow of checking a list of URLs for target elements, with a split-screen view of the browser window. 

To access the **voice control accessible version**, visit the voice branch here: https://github.com/ddamico-at-thermo/quick-audit/tree/voice

Note that you will need to download and locally run the voice version, as there is no active GitPages deployment at this time.
